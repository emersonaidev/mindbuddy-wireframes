MindBuddy Screens - Ready for Figma Import

How to import to Figma:

  1. Install "html.to.design" plugin in Figma
  2. Click on any screen below to open it
  3. Select all (Cmd+A) and copy the HTML
  4. Paste into the plugin
  5. Click "Import to Figma"
